# DX4MGR Ver5 変更点一覧 (from Ver4)

Ver5では、シミュレーションの「精度」と「確信度」を高めるための大幅な構造変更を行いました。

## 1. シミュレーションエンジンの改良
- **並列実行の導入**: モンテカルロ試行を `multiprocessing` で実行するように変更。実行速度がコア数に応じて数倍高速化されました。
- **生データ収集の強化**: 解析のために、全試行のサマリー値を配列で保持するように拡張しました。

## 2. 解析ロジック (Analyzer) の新設
- **検証ゲート (Quality Gates)**: 各シナリオ実行後に「合格/不合格」を判定するロジックを新設。
- **統計解析機能**:
  - 95%信頼区間の自動算出。
  - Baseline比較における「有意差」の自動判定。
- **DOE (実験計画法)**:
  - `Latin Hypercube Sampling` による自動パラメータ探索機能を搭載。

## 3. 可視化 (Visualizer) の強化
- **統計情報の追加**:
  - スループットの棒グラフにエラーバー（95%CI）を追加。
  - 納期超過確率 (CCDF) のグラフを正式搭載。
- **新グラフ**:
  - `DOE分析図`: パラメータと結果の相関。
  - `Quality Gateステータス図`: 各判定ゲートの成否を一覧表示。

## 4. 実行フロー (Pipeline) の刷新
- **実験主導型フロー**:
  1. まずDOEでパラメータの当たりをつける。
  2. メインシナリオを実行。
  3. 検証ゲートで品質をチェック。
  4. 統計的に比較。
  5. 結果をJSONレポートとして出力。
