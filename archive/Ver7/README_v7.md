# DX4MGR Ver7: パス解決・ドキュメント刷新モデル

## 概要
Ver7では、Ver6で発生していた実行環境依存のパス問題を解消し、どのようなディレクトリからでも安定して実行できるように再構築しました。
また、実務での利用を想定し、日本語ドキュメント（初心者ガイド、パラメータ解説、分析のヒント）を大幅に拡充しています。

## 主な変更点
1. **パス解決の堅牢化**: スクリプトの場所を基準に相対パスを絶対パスへ変換するロジックを導入。`scenarios.csv` が見つからないエラーを修正。
2. **コードの整合性向上**: 全てのモジュールを Ver7 体系に統一。
3. **ドキュメントの完全日本語化**: 初心者でも迷わず解析を開始できるガイドを追加。

## ファイル構成
- `run_v7.py`: メイン実行スクリプト。
- `simulator_v7.py`: シミュレーションロジック（差し戻し負荷考慮）。
- `analyzer_v7.py`: 統計解析・検証ゲート判定。
- `visualizer_v7.py`: グラフ可視化。
- `scenarios.csv`: 解析シナリオの設定。
- `DOCS/`: 日本語ドキュメント類（新規・刷新）。

## 実行方法
```bash
python3 2026/RDIssues/Ver7/run_v7.py
```
どのディレクトリから実行しても、自動的に `2026/RDIssues/Ver7/output/` に結果が出力されます。

## 解析結果の見方
1. `v7_step1_doe_analysis.png`: パラメータ感度分析。
2. `v7_step2_gate_*.png`: 各シナリオの目標達成判定。
3. `v7_step3_comparison_ci.png`: シナリオ間の統計的比較。
